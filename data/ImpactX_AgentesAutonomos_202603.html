Empresas LangGraph

Frameworks Code-First

Controle total, escrevendo codigo. Ideal para agentes complexos e em producao.

LangGraph

Grafo de estados + checkpointing

Arquitetura: Grafo direcionado com nos e arestas. Visibilidade total do caminho de execucao.

Diferencial: Checkpoint rollback, observabilidade via LangSmith. Uber, LinkedIn, BlackRock em prod.

Modelos: Qualquer LLM | Curva: 1-2 semanas

v1.0 GA ~25K stars

CrewAI

Multi-agente baseado em papeis

Arquitetura: Agentes como "membros de time" com role, backstory e goal.

Diferencial: 20 linhas Python = sistema funcionando. MCP + A2A nativo. 100K+ devs certificados.

Modelos: Qualquer LLM | Curva: 3-5 dias

v1.10.1 44.6K stars

Claude Agent SDK

Tool-use + MCP nativo

Arquitetura: Tool-use chain com sub-agents. MCP in-process server model.

Diferencial: MCP criado pela Anthropic. Constitutional AI. Extended thinking transparente.

Modelos: Claude only | Curva: Medio

v0.1.48 MCP-native

OpenAI Agents SDK

Handoff chain minimalista

Arquitetura: Agents delegam para especialistas como tools. SDK gerencia loops automaticamente.

Diferencial: <100 linhas = multi-agent com guardrails e tracing built-in.

Modelos: OpenAI only | Curva: 2-3 dias

v0.10.2 OpenAI-only

Google ADK

Multi-modal + A2A protocol

Arquitetura: Code-first (Python, Java, Go). A2A para comunicacao cross-framework.

Diferencial: Gemini 2.5 Pro multi-modal. Deploy Cloud Run com auto-scaling.

Modelos: Gemini + Claude | Curva: Medio

v1.26.0 A2A

OpenClaw

Multi-agente config-driven

Arquitetura: JSON + Markdown config. Gateway localhost:18789. Workspaces isolados.

Diferencial: Budget caps por agente. EcoMode. Circuit breaker. Cron nativo. Composio 500+ tools.

Modelos: Multi (routing) | Curva: Config-heavy

Prod 24/7 Oracle VPS

Plataformas No/Low-Code e SaaS

n8n

Workflow automation + AI agents

Plataforma low-code com 500+ integracoes. Valuada em $2.5B (Oct 2025). 5,815+ templates de automacao AI.

WhatsApp: Integracao nativa via Evolution API ou Cloud API oficial.

Custo: Self-hosted gratis. Cloud $40-250+/mes.

Prod WhatsApp nativo

Perplexity Computer

19 modelos + execucao autonoma

"Digital worker" que orquestra 19 modelos AI em paralelo. Workflows de horas a meses. 400+ integracoes.

Custo: $200/mes. Limitacao: Sem WhatsApp. Foco enterprise.

Prod $200/mes

Manus AI

Agente autonomo generalista

Planeja, navega, codifica e analisa em background. Adquirido pela Meta. Usa Claude e Qwen internamente.

Custo: Free (300 creditos/dia), $39-199/mes. Problema: Instabilidade significativa.

Instavel Custo imprevisivel

Claude Code + Cowork

IDE-integrated + Agent Teams

340+ plugins, 1367+ skills. Agent Teams (Feb 2026). Subagents para tarefas paralelas.

Limitacao: Requer sessao ativa. Nao e proativo/scheduled por design.

Prod Sessao required

Composio

Integration layer para agentes

500+ tools pre-prontas com OAuth, rate-limit e retry gerenciados. MCP nativo. Funciona com qualquer framework.

Custo: Free/freemium. Ja integrado no OpenClaw.

Prod Ja integrado

Conectando Agentes ao WhatsApp

Meta baniu chatbots generalistas no WhatsApp (Jan 2026). Porem, bots especificos para atender clientes/usuarios (como agente de saude pessoal) continuam permitidos.

3 Caminhos

1. Evolution API + n8n

RECOMENDADO

Open-source. Dual-mode: Baileys (gratis) ou Cloud API (producao). 10+ integracoes. n8n community node pronto.

Multi-modal: texto, audio (transcricao), imagem (visao).

Templates n8n prontos:

  • WhatsApp assistant GPT-4o + Evolution API
  • Agent "human-like" com Redis + PostgreSQL + Gemini
  • AI personal assistant com RAG + voice via Supabase

2. Baileys Direto (TypeScript)

PARA DEVS

Protocolo WebSocket do WhatsApp Web direto. 100% gratis, Linked Devices, controle total.

Riscos: Nao oficial, conta pode ser banida, sem SLA.

Melhor para: Projetos pessoais, uso proprio.

3. WhatsApp Cloud API (Oficial)

PRODUCAO

API oficial da Meta. 1.000 conversas gratis/mes. Templates aprovados. Webhooks nativos.

Requer: Business Account verificado. Custo por conversa apos free tier.

Alternativa: Twilio — mais facil, porem mais caro.

Stack Recomendado

Uso pessoal (custo zero):
WhatsApp (Baileys/Evolution API) → n8n → Claude/Gemini Flash → Supabase → Cron
Producao/escala:
WhatsApp Cloud API → Evolution API → n8n → LLM → PostgreSQL → Cron

Guardrails (AuDHD-aware)

  • Max 1 pergunta por mensagem — nunca bombardear
  • Max 5 itens por lista — respeitar working memory
  • Respostas curtas — 2-3 frases max
  • Sem julgamento — nunca "voce deveria ter..."
  • Comando /pausar — para tudo imediatamente
  • Protocolo de crise — CVV 188 + contato emergencia hardcoded
  • Nao diagnosticar — sempre redirecionar para profissional

Recomendacao Final

Caminho recomendado: Hibrido

OpenClaw (ja roda 24/7) como orchestrador + memoria + guardrails
+ Evolution API (gratis, self-hosted) como canal WhatsApp

+ Claude Haiku (fallback) para nuance em reestruturacao cognitiva
