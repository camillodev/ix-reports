ix-video — Content Robot CLI

Automated video content creation and processing pipeline using yt-dlp, Claude AI, and FFmpeg.

Project Overview
techautomacaoestrategia

A comprehensive CLI tool for automating video content creation, editing, and distribution across multiple platforms and languages.

Key Technologies
  • yt-dlp: Advanced video downloading and processing
  • Claude API: AI-powered content generation and analysis
  • FFmpeg: Professional video encoding and transformation
  • Seedance: Distribution and scheduling framework

Architecture

3 Fases (Phases)
Phase 1: Ingestion

Video source detection and download using yt-dlp. Multi-source support including YouTube, local files, and streaming endpoints.

Phase 2: Processing

Claude AI-powered analysis, content generation, and FFmpeg-based transformations. Intelligent scene detection and segment optimization.

Phase 3: Distribution

Seedance-based scheduling and distribution across multiple platforms with format optimization and metadata generation.

3 Modos (Modes)
Mode 1: Automated

Full pipeline automation with intelligent defaults and minimal user intervention required.

Mode 2: Interactive

Step-by-step guidance with human approval checkpoints at critical stages.

Mode 3: Manual Override

Fine-grained control over each processing stage with advanced parameter tuning.
