Como criar cada view
  1. Va em Contacts (ou Deals)
  2. Clique "Add filter" e selecione o campo/valor indicado
  3. Clique "Save view" → nomeie conforme a tabela
  4. Ajuste as colunas: clique "Edit columns" e adicione as colunas listadas

Automacoes HubSpot (Starter Plan)

O Starter plan permite automacoes simples de tasks por stage change. Automacoes mais complexas vao no n8n (aba seguinte).

Onde configurar Automation → Workflows → Create workflow → Deal-based

Workflow 1 — Task: Prep Call

ConfigValor
TriggerDeal enters stage "Discovery Call"
ActionCreate task
Task title"Prep Call — " + Deal name
Assigned toDeal owner (Pimenta)
Due date1 day from trigger

Workflow 2 — Task: Send Proposal

ConfigValor
TriggerDeal stage changes from "Discovery Call" to "Proposal Sent"
ActionCreate task
Task title"Follow-up proposal — " + Deal name
Assigned toDeal owner
Due date3 days from trigger

Workflow 3 — Task: Stale Negotiation Alert

ConfigValor
TriggerDeal enters stage "Negotiation"
Action 1Delay 10 business days
Action 2If/then: deal STILL in "Negotiation"
Action 3Create task: "Stale deal alert — " + Deal name
Assigned toDeal owner + Rafa
Limitacao do Starter O Starter plan nao suporta "If/then" branches complexos. Se a Workflow 3 nao funcionar, simplifique: apenas crie a task apos delay de 10 dias.

6 Workflows n8n

Automacoes mais poderosas que rodam no n8n (gratis, ja configurado). Substituem a necessidade do HubSpot Professional ($90/mo).

Vantagem n8n vs HubSpot Pro n8n e gratis, permite integracao com Linear, Gmail, Slack, Claude. HubSpot Professional custa $90/mo e nao integra com nossas ferramentas.
