O que e Samantha?

Samantha e a assistente AI pessoal e corporativa do Rafael Camillo, fundador da Impact X. Ela esta disponivel 24 horas por dia, 7 dias por semana, via texto e voz, conectada a todos os dados pessoais e da empresa com controles rigorosos de budget e seguranca.

Inspiracao: Assim como no filme "Her" (2013), Samantha nao e apenas um chatbot — e uma companheira inteligente que entende contexto, toma iniciativa e evolui com o usuario.

Por que agora?

A infraestrutura ja existe: OpenClaw rodando na VPS Oracle (24GB RAM), 14+ MCPs configurados (Linear, Notion, Gmail, Calendar, GitHub, Vercel, Supabase), Finance Agent ativo, Silva no Slack, Composio para 500+ apps. Falta unificar tudo numa experiencia coesa.

Decisao Arquitetural

OpenClaw "sam" + Bridges Custom — O agente "sam" ja esta bootstrapped no OpenClaw com workspace, sessions, heartbeat e multi-LLM routing. Construir bridges TypeScript para WhatsApp, voice e RAG. Nao reinventar a roda.
AlternativaProsContrasDecisao
100% OpenClaw Tudo integrado WhatsApp bridge nao existe no framework Parcial
100% Custom (Fastify) Controle total Reescrever sessions, memory, heartbeat Rejeitado
Hibrido (escolhido) Reusa OpenClaw + bridges custom Duas camadas para manter Aprovado

Fluxo de uma Mensagem

1. Entrada

Rafael envia "o que tenho na agenda amanha?" no WhatsApp.

2. Gateway

Baileys recebe → Fastify valida auth (numero do Rafael) → Budget Gate estima custo → Forward para OpenClaw gateway HTTP (localhost:18789).

3. Orquestracao

Intent Classifier (Ollama, custo $0) identifica: calendar.list_events. LLM Router seleciona Gemini Flash ($0.075/1M). Session Manager carrega contexto.

4. Execucao

MCP Google Calendar lista eventos de amanha. LLM formata resposta natural.

5. Resposta

Texto volta pelo gateway → WhatsApp Bridge → Rafael recebe no celular. Custo logado no budget_ledger. Acao logada no audit_log.

Budget Control & Security

Sistema de Controle de Budget

O Budget Gate intercepta toda request antes de chegar ao LLM. Estima custo, checa limites, faz downgrade de modelo se necessario, e bloqueia se o budget estourou.

Por Request
$0.50
Rejeita se estimativa exceder
Diario
$5.00
Alerta em $4.00
Semanal
$25.00
Alerta em $20.00
Mensal
$100.00
Alerta em $80.00
Estimativa mensal realista: ~$30-50 com routing agressivo para Ollama (50% das requests, custo zero) e Gemini Flash (35%, $0.075/1M input).

Tabela budget_ledger

CREATE TABLE sam_budget_ledger (
  id          UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at  TIMESTAMPTZ DEFAULT now(),
  model       TEXT NOT NULL,        -- 'ollama:qwen2.5:7b', 'gemini-flash', etc.
  input_tokens  INT NOT NULL,
  output_tokens INT NOT NULL,
  cost_usd    DECIMAL(10,6) NOT NULL,
  intent      TEXT,                 -- 'calendar', 'email', 'chat', etc.
  channel     TEXT,                 -- 'whatsapp', 'slack', 'heartbeat'
  session_id  TEXT
);

Modelo de Seguranca

Classificacao de Dados

PUBLIC — Docs publicos, repos open-source, informacoes gerais. Acesso livre.
INTERNAL — Linear issues, Notion pages da empresa, GitHub privado. Leitura auto-aprovada.
CONFIDENTIAL — Email, Calendar, transacoes financeiras. Leitura auto-aprovada, escrita com confirmacao.
SECRET — .env, SSH keys, API keys, senhas. NUNCA acessar ou expor. Hard block.
