Estudo de Caso: Viktor
Modelo de proatividade que devemos replicar no P1

Viktor (getviktor.com) é um "AI coworker" que vive no Slack. Não é um chatbot: é um agente autônomo com sandbox cloud próprio que executa ações reais.

💓 O Heartbeat System (Diferencial)

  1. Observa canais do Slack continuamente
  2. Analisa padrões de uso das ferramentas conectadas (Google Ads, Stripe, HubSpot, etc)
  3. Detecta anomalias via clustering não-supervisionado
  4. Propõe automações que ninguém pediu
  5. Uma vez aprovada, roda em autopilot
Exemplo real: Durante crise, Viktor postou 28 updates automáticos, rastreou voos da equipe, flaggou anomalia de spend em ads, e shippou um PR — tudo sem ser solicitado.

🧠 Self-Improving Skills

Viktor documenta workflows que executa e os transforma em skills reutilizáveis. Quanto mais usa, mais automatiza.

🔧 Modelo de Execução

  • Sandbox cloud isolado por workspace (TypeScript)
  • Browser automation via Puppeteer
  • Deliverables reais: PDFs, Excel, PowerPoint, web apps com DB
  • 3000+ integrações via OAuth gerenciado

⚖️ Regra Meta 2026 — Critical para P1

Meta agora exige que chatbots realizem "tarefas concretas de negócio".

Chatbots open-ended sem objetivo claro não são permitidos.

Um agente terapêutico precisa ser enquadrado como "suporte à saúde" com tarefas específicas:

  • Gerenciamento de rotina
  • Lembretes de medicação
  • Exercícios de CBT estruturados

🎯 Estratégia Recomendada para P1

  1. Produção: Evolution API (Cloud API) — compliance, sem risco de ban, custo controlado ($0.03-0.15/msg)
  2. Prototipagem: Evolution API (Baileys) para MVP — rápido, grátis, risco aceitável em testes
  3. Futuro: Considerar Twilio se precisar multi-canal (SMS, voice, etc)

🧠 Design do Agente Terapêutico (Pattern)

Rotinas Proativas (cron):
  • 08:00 — Check-in matinal: "Como você dormiu? Como está o nível de energia?"
  • 12:00 — Lembrete de pausa/alimentação (ADHD esquece de comer)
  • 18:00 — Revisão do dia: "O que você conseguiu fazer hoje?" (sem julgamento)
  • 22:00 — Rotina de wind-down: exercício de respiração ou journaling guiado
Triggers Reativos:
  • Detecção de dysregulação emocional → protocolo de grounding (5-4-3-2-1)
  • Muitas mensagens rápidas → "Parece que você está acelerado. Quer fazer um exercício de 2 min?"
  • Silêncio prolongado (>24h) → check-in gentil
Memória Persistente:
  • Padrões de humor ao longo do tempo
  • Rotinas que funcionam vs não funcionam
  • Triggers pessoais identificados
  • Progresso em objetivos terapêuticos

📊 Métricas de Sucesso (4 semanas)

  • Usuários ativos diários > 50%
  • Engagement rate (check-ins respondidos) > 70%
  • NPS > 7 (recomendaria a amigos?)
  • Custo por usuário < $0.50/mês
