🔴 P0 — Semana 1-2 (eliminar blockers)

1. Fechar 3 primeiros deals por referral
P0HUMANO$0

Custo
$0 — só tempo
Mitiga

2. Construir bench de 10 senior devs BR
P0HÍBRIDO~$200/mês
Sem bench, não escala. Precisa ter 10 devs verificados e prontos pra alocar quando o deal fechar.
Ação humana
Rafa faz entrevista técnica final (30min cada). Nicole gerencia pipeline de candidatos.
Ação IA
Claude faz screening inicial de CVs, analisa GitHub/LinkedIn, gera relatório de fit por stack. n8n automatiza coleta de candidatos de fontes (LinkedIn, comunidades BR).

Custo

Mitiga

3. Site IX com cases e social proof
P0HÍBRIDO~$50/mês

Ação humana

🤖 Soluções IA — Detalhamento

A IA é o multiplicador que permite 3 pessoas competirem com empresas de 100-300. Cada solução abaixo substitui horas de trabalho manual ou uma contratação.

Stack de IA da Impact X Claude (Max/API) → cérebro. n8n (self-hosted) → automação/orquestração. Supabase → dados. Evolution API → WhatsApp. Hono + Maestro → API gateway (OpenClaw).
Solução IA #1
🔎 Auto-Research Engine
O que faz
Recebe nome de empresa → pesquisa automaticamente: funding (Crunchbase), tamanho (LinkedIn), stack (BuiltWith/StackShare), decisor (LinkedIn) → preenche HubSpot → gera Fit Check score.
Benefício
30min/lead → 2min/lead. Rafa/Nicole pesquisam 3 leads/dia manualmente. Com IA: 15-20 leads/dia. 5x mais pipeline com mesmo tempo.
Como resolver
n8n workflow: trigger = novo Contact no HubSpot → Claude API (web search) → parse dados → update Contact fields → calcular ix_fit_score → notificar Rafa se score alto.
IA usada
Claude API (Sonnet pra custo, Opus pra análise complexa)
Custo
~$0.05/lead (Claude API). 20 leads/dia = $1/dia = $30/mês.
Timeline
1 semana pra build. n8n + Claude API já no stack.
Substitui
~1 SDR part-time ($1,500-2,000/mês)
Solução IA #2
📋 Proposal Generator
O que faz
Após discovery call, Claude pega os dados do deal (dor, perfis, budget) → gera proposta branded completa via ix-reports → Rafa/Pimenta revisam → enviam no mesmo dia.
Benefício
2-3 horas/proposta → 15 minutos review. Proposta sai no dia da call. Concorrentes levam 2-5 dias. Startup fecha com quem responde primeiro.
Como resolver
Template parametrizado no ix-reports. n8n trigger: deal muda pra "Proposal Sent" → Claude API gera HTML com dados do deal → salva em ix-reports → notifica pra review.
IA usada
Claude API (Opus pra qualidade de escrita)
Custo
~$0.50/proposta (Opus). 10 propostas/mês = $5/mês.
Timeline
3 dias pra build (template + workflow).
Substitui
~4-6 horas/semana de trabalho manual de proposta
Solução IA #3

Benefício

Como resolver
n8n: Contact status = "Qualified" → Claude API gera draft → salva como nota no HubSpot → task pra Nicole revisar e enviar.
IA usada
Claude API (Sonnet — bom o suficiente pra emails)
Custo
~$0.02/email. 100 emails/mês = $2/mês.
Timeline
2 dias pra build.
Substitui
~1 SDR dedicado a outreach ($2,000/mês)

Solução IA #6
🤖 Samantha — WhatsApp Qualifier
O que faz
Agente WhatsApp com personalidade natural (inspirada em "Her") que qualifica inbound leads com perguntas conversacionais. Se for ICP, cria deal no HubSpot e notifica Pimenta.
Benefício
Qualificação 24/7 sem ação humana. Lead manda mensagem às 23h → Samantha qualifica → deal criado → Pimenta acorda com lead quente na fila.
Como resolver
Já em build no OpenClaw. Evolution API (WhatsApp) → Hono → Maestro (Opus) → Haiku classifier + generate pattern → Supabase.
IA usada
Haiku (classificação) + Sonnet (geração de resposta)
Custo
~$10-12/mês (estimativa do pattern two-step)
Timeline
Em build. MVP em 3-4 semanas.
Substitui
~1 SDR part-time pra triagem de inbound
Solução IA #7
👤 CV Screening + Dev Matching
O que faz
Recebe CV/LinkedIn de dev candidato → Claude analisa: stack match, seniority, English level, GitHub activity → gera score + relatório → Rafa só entrevista os top scored.
Benefício
20 CVs/hora vs. 5 CVs/hora manual. Rafa só investe tempo entrevistando devs que já passaram no filtro IA.
Como resolver
n8n: novo candidato no pipeline → Claude API analisa CV + LinkedIn → score 1-10 → se ≥7, agenda entrevista com Rafa.
IA usada
Claude API (Sonnet)
Custo
~$0.10/CV. 100 CVs/mês = $10/mês.
Timeline
3 dias.
Substitui
~8 horas/mês de screening manual
