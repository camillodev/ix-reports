4 Modelos de Receita Concretos

Cada modelo usa assets que ja existem. Nenhum exige desenvolvimento de produto novo — apenas empacotamento e venda.

Modelo 1

Prospeccao como Servico (Agent-as-a-Service)

Vender o pipeline Olheiro → Garimpeiro → Grio → Mare como servico para outros founders e consultorias que precisam prospectar no mercado US.

Como funciona

  1. Cliente define ICP (perfil de cliente ideal) + copy base
  2. Voce configura os agentes com os filtros dele no Apollo/LinkedIn
  3. Agentes rodam semanalmente: prospectam, qualificam, geram outreach, fazem follow-up
  4. Cliente recebe deals quentes no HubSpot dele (ou planilha)

Precificacao

ComponenteValorTipo
Setup inicialR$ 2.000 — 5.000One-time
Mensalidade (ate X leads qualificados)R$ 1.500 — 3.000/mesRecorrente
Modelo performanceR$ 500/lead qualificadoPor resultado
Por que funciona: Founders de consultoria/agencia BR que vendem para US tem o mesmo problema — pipeline frio. Voce ja resolveu tecnicamente, so precisa empacotar.

Timeline para primeira venda: 2-4 semanas

Pegar projetos curtos de automacao usando seu stack (n8n + Claude + Composio + agentes) para empresas que precisam de integracao AI.

  • Comunidades n8n/Make — muita demanda, pouca oferta de quem sabe Claude

  • Rede pessoal — founders BR que vendem para fora
